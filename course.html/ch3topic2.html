Digital Logic and Circuit Design

Combinational Circuits: Adders, Multiplexers, Decoders

Combinational circuits are digital circuits whose output depends solely on the current inputs, with no memory or feedback. They perform a wide range of logic operations and are fundamental building blocks in digital systems. Three common types of combinational circuits are adders, multiplexers, and decoders.

1. Adders

Adders are circuits that perform arithmetic addition of numbers, usually in binary form. There are different types of adders based on the number of bits involved and the complexity of the circuit.

Types of Adders:

  • Half Adder (HA): A half adder adds two single-bit numbers and produces a sum and a carry-out.
  • Full Adder (FA): A full adder adds three input bits and produces a sum and a carry-out.
  • Ripple Carry Adder (RCA): A chain of full adders used for multi-bit binary addition.
  • Carry Lookahead Adder (CLA): An optimized full adder that speeds up addition by calculating carry values in parallel.

2. Multiplexers (MUX)

A multiplexer is a digital switch that selects one of several input signals and forwards the selected input to a single output line.

Example: 2-to-1 MUX

  • Inputs: D0, D1, S (selection input)
  • Output: Y
  • Truth Table:
    • When S = 0, Y = D0
    • When S = 1, Y = D1

3. Decoders

A decoder takes an n-bit binary input and activates one of 2^n outputs corresponding to the binary input.

Example: 2-to-4 Decoder

  • Inputs: A1, A0
  • Outputs: O0, O1, O2, O3
  • Truth Table:
    • When A1A0 = 00, O0 = 1, others = 0
    • When A1A0 = 01, O1 = 1, others = 0
    • When A1A0 = 10, O2 = 1, others = 0
    • When A1A0 = 11, O3 = 1, others = 0
