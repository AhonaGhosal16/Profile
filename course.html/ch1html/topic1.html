Overview of Computer Systems

Components of a Computer System

Hardware

Hardware refers to the physical parts of a computer that you can touch and see. This includes components like the CPU, memory, hard drive, keyboard, and monitor.

  • Central Processing Unit (CPU): The brain of the computer that performs calculations and executes instructions. It consists of:
    • Arithmetic Logic Unit (ALU): Performs mathematical and logical operations.
    • Control Unit (CU): Directs the operation of the processor and coordinates activities.
  • Memory: Includes RAM (Random Access Memory), which temporarily stores data currently in use, and ROM (Read-Only Memory), which stores essential instructions for starting the computer.
  • Storage Devices: Hard Disk Drives (HDD) and Solid-State Drives (SSD) retain data even when the power is off.
  • Input Devices: Tools like keyboards, mice, and scanners for entering data.
  • Output Devices: Components like monitors, printers, and speakers that convey information to the user.

Software

Software refers to programs and instructions that tell the hardware what to do. This includes operating systems (e.g., Windows) and application software (e.g., word processors, games).

  • Operating System (OS): The fundamental software that manages hardware resources and facilitates communication between hardware and applications.
  • Application Software: Programs designed for specific tasks, such as word processors and web browsers.

Difference between Hardware and Software

Aspect Hardware Software
Definition Physical components of a computer Programs and instructions for the computer
Tangible Can be touched and seen Intangible, cannot be physically touched
Function Performs tasks through physical processes Tells the hardware what tasks to perform
Examples CPU, RAM, Hard drives Operating systems, applications
Installation Installed physically (plugging in devices) Installed digitally (downloading)
Maintenance Requires physical repairs Requires updates, bug fixes
